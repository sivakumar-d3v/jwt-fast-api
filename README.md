# jwt-fast-api


## Project Setup

#### 1. Clone the code on machine
```bash
  git clone  https://github.com/sivakumar-d3v/jwt-fast-api.git
```

#### 2. Create python environment

```bash
  cd jwt-fast-api
  python -m venv venv
```

#### 3. Activate source 

```bash
 source bin/activate
```

#### 4. Run Application
```bash
  uvicorn app.main:app --reload --port 8000
```
